📖 사용자 가이드

🎬 3단계 워크플로우

  1. 미디어 준비: 오디오 + 배경(이미지/단색/그라데이션/애니메이션) + 로고 업로드 + 인코딩 설정
  2. 비주얼 편집: 8장르 프리셋, 15+ 스펙트럼, 18 이펙트, 9 타이틀 스타일, 가사 3줄/롤링
  3. 영상 출력: WebCodecs MP4 직접 렌더링 → 다운로드

🎵 가사 LRC 자동 정리

"📂 LRC/TXT 파일 열기"로 업로드하면 [Intro] 같은 섹션 태그 제거 + 단어 단위→문장 병합이 자동 적용됩니다. 평문 TXT도 OK.

🎨 가사 표시 모드

  • 단일: 현재 줄만
  • 3줄: 이전·현재·다음 (노래 따라 읽기 좋음)
  • 전체: 9줄 롤링 뷰

📍 타이틀 위치

6-위치 그리드(좌상단/상단중간/우상단/좌아래/중간아래/우아래) + 미세 X/Y로 가사와 겹치지 않게 자유 배치

⚡ 성능 모드

  • 풀질: 모든 효과 최대치
  • 자동: 효과 많이 켜면 자동 감속
  • 성능: 무거운 효과(⚡) 자동 OFF + 파티클 1/3

🖱️ 단축 키 / 팁

  • 배경 썸네일을 드래그해서 순서 변경
  • 배경 카드의 A↔Z 버튼: 이름순 정렬 (1번=오름, 2번=내림, 3번=원래)
  • 새로고침해도 모든 설정·파일이 유지됨 (IndexedDB)
  • 전체 초기화: 사이드바 하단 ↻ 전체 초기화 또는 Stage 1 하단 ↻ 초기화

🌐 환경 요구

Chrome/Edge 최신 버전 (WebCodecs H.264 지원 필요). 인터넷 연결 필요 (mp4-muxer CDN).
